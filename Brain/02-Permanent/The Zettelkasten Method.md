# The Zettelkasten Method

The Zettelkasten, or slip-box, is a note-taking method in which many small notes are written, given unique identifiers, and densely linked to one another. Knowledge accumulates through the connections between notes rather than through a fixed filing scheme.

The method has three working parts: fleeting notes for quick capture, literature notes that summarize sources, and permanent notes that record durable ideas in one's own words. Each permanent note is linked to related notes so that following a chain of links can surface unexpected combinations. Because notes are addressed by identity and reference rather than by location in a folder tree, the same note can participate in many lines of thinking at once. The result is a system that grows more useful as it grows larger, since each new note can connect to everything already present.

**Why this matters**: It demonstrates that a knowledge base gains power from its links, not from how neatly its files are sorted.
