# Progressive Summarization Distills Notes Over Time

Progressive summarization is a technique for compressing notes in layers, highlighting the most important passages on each successive pass. A note is refined gradually so that its essence becomes easy to grasp at a glance later.

The method works in stages: first capture the source material, then on a later review highlight the key sentences, and on a further pass bold or restate the few phrases that matter most. Each layer is added only when the note is revisited, so effort is spent on material that proves useful rather than on everything at once. The goal is to make a note's core idea instantly accessible to a future reader without discarding the fuller context beneath it. Distilling on revisit also deepens engagement, since deciding what to keep requires understanding what the note actually says.

**Why this matters**: It keeps a growing note collection usable by ensuring each note can be understood quickly when it is needed again.
