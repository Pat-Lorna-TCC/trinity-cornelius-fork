# Value Lives in Changes from a Reference Point - Bernoulli's Error

**Source**: Daniel Kahneman, *Thinking, Fast and Slow* (Farrar, Straus and Giroux, 2011) - Part IV, "Bernoulli's Errors" / "Prospect Theory"
**Document Type**: Book
**Extracted By**: AI (document-insight-extractor, Book Mode)
**Extraction Date**: 2026-06-28
**Scope**: Books/thinking-fast-and-slow-kahneman

## Core Insight
The carrier of subjective value is the CHANGE - a gain or loss relative to a reference point - not the absolute end-state of wealth. This is the variable Bernoulli's 300-year-old expected-utility theory omitted, and the omission is why utility theory fails to describe real choice.

## Mechanism / Why It Matters
The prospect-theory value function packs three features into one S-shaped curve: (1) reference-dependence - value is measured from a baseline, not from zero wealth; (2) diminishing sensitivity - the curve is concave for gains and convex for losses, which mechanically produces risk-seeking in the domain of losses; and (3) loss aversion - the loss side is steeper than the gain side. Fixing the reference point is therefore the master move from which the rest of the architecture follows.

## Evidence
Jack and Jill both end at $5M (Jack rose from $1M, Jill fell from $9M). Utility theory predicts identical happiness; Jack is elated, Jill miserable. The three-bowls-of-water demo shows reference-dependence directly. Diminishing sensitivity: $100->$200 feels larger than $900->$1,000, the same logarithmic compression Fechner found for brightness and loudness. Markowitz proposed reference-dependence ~25 years earlier and was ignored.
