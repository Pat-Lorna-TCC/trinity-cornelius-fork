# Decorrelate Errors - Independent Judgments Beat Correlated Ones
**Source**: Daniel Kahneman, *Thinking, Fast and Slow* (Farrar, Straus and Giroux, 2011) - Part 3 / judgment-aggregation discussions
**Document Type**: Book
**Extracted By**: AI (document-insight-extractor, Book Mode)
**Extraction Date**: 2026-06-28
**Scope**: Books/thinking-fast-and-slow-kahneman

## Core Insight
The wisdom of crowds works ONLY when individual errors are independent. Any shared influence - open discussion, sequence effects, first-speaker dominance - correlates errors and destroys the aggregation benefit. To extract real information from multiple sources, you must actively FORCE their independence before combining them.

## Mechanism / Why It Matters
Averaging cancels error only if errors point in random directions. The moment judges influence one another, their errors align, and the average inherits the shared bias instead of cancelling it - more judges then add false confidence rather than accuracy. The actionable rule: collect judgments independently first, aggregate second. The transferable error this guards against is mistaking a confident consensus for many independent confirmations when it is really one opinion echoed.

## Evidence
Kahneman's grading reform: scoring exams question-by-question across all booklets, instead of grading each booklet whole, broke his own halo effect and revealed the true variance in students' grades. Police keep eyewitnesses apart precisely to preserve error independence. The meeting rule: have everyone write down their position BEFORE discussion begins, because open discussion overweights whoever speaks first or loudest and silently correlates the room.
