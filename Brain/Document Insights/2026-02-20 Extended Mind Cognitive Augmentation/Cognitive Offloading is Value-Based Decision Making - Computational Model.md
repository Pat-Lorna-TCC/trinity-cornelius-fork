## Core Framework

Cognitive offloading is not a failure of willpower or a sign of laziness - it is a **rational, value-based optimization** of a limited-resource system. The decision to offload follows the same computational structure as any value-based choice: weighing expected costs against expected benefits.

**The model's two core principles**:
1. Items stored in brain-based memory occupy limited capacity, generating an **opportunity cost**
2. External reminders incur only a small **physical-action cost** with effectively unlimited capacity
3. These costs are balanced against the **value of remembering** - yielding an optimal offloading strategy

## Context and Acceptance

**Field Acceptance**: Emerging consensus in cognitive science; aligns with broader value-based decision-making literature
**Significance**: First rigorous computational account of offloading decisions; formalizes extended mind behavior in decision theory

**Six Phenomena the Model Successfully Predicts**:
1. Preferential offloading of high-value items (we save what matters most)
2. Increased offloading at higher memory loads (when internal capacity is strained)
3. Forgetting of offloaded items ("Google Effect") - reduced internal encoding
4. Improved memory for non-offloaded items ("saving-enhanced memory") - freed capacity goes to retained items
5. Reduced saving-enhanced memory when reminders are unreliable (trust mediates encoding decisions)
6. Increased offloading in individuals with poorer memory ability (rational compensation strategy)

## The Rational Offloading Thesis

The question is not whether to offload but **which cognitive processes represent rational offloading versus harmful dependency**:

**Rational offloading** (procedural, retrieval-based tasks):
- Opportunity cost of internal storage is high relative to value
- External storage is reliable and accessible
- Minimal learning benefit from performing internally
- Examples: phone numbers, appointment times, routine calculations

**Potentially harmful offloading** (integrative reasoning, synthesis):
- The value comes not just from having the output but from the neural process of generating it
- Offloading eliminates the very learning that makes internal capacity expand
- The "saving-enhanced memory" effect disappears - nothing is encoded by the human
