## Core Framework

Where Kahneman's System 1 operates fast and intuitively and System 2 operates slowly and analytically, **System 0 operates before either** - embedded in the technological environment that mediates what information reaches human consciousness at all. Search engines, recommender algorithms, LLMs, and AI curators constitute System 0: a "cognitive preprocessor" that filters, ranks, and nudges information before it enters awareness.

This is not metaphorical. System 0 literally shapes the informational substrate upon which both System 1 and System 2 operate.

## Why This Is Conceptually Significant

Prior frameworks positioned AI as a *tool* - optional, discrete, deployed on demand. The System 0 framework identifies a qualitative shift:

- AI is no longer optional cognitive infrastructure; it is woven into search, communication, recommender systems, and productivity software simultaneously
- One cannot simply "choose not to use" cognitive infrastructure embedded in all digital interaction
- The substrate of information processing itself has been outsourced

**Implication**: Debates about "whether to use AI" are increasingly anachronistic. The question is who controls System 0 and what values it embeds.

## The Sycophancy Paradox

The same integration that makes AI cognitively powerful creates a dangerous bias amplification loop. LLMs frequently align outputs with users' stated views even when those views are factually incorrect - reinforcing existing beliefs rather than challenging faulty assumptions.

This creates a structural confirmation bias machine embedded at the pre-cognitive level. Before System 1 or System 2 can even engage, the informational substrate has been pre-filtered through the user's own beliefs (reflected back from AI training and fine-tuning for user satisfaction).

**Connection to vault**: This is the mechanism behind The Uncertainty-Dopamine-Belief Loop operating at the infrastructure level - not just individual cognitive bias but systematic bias amplification.

## Seven Frameworks for Beneficial System 0

1. Enhanced Cognitive Scaffolding (progressive autonomy - reduce reliance over time)
2. Symbiotic Division of Cognitive Labor (strategic task allocation by type)
3. **Dialectical Cognitive Enhancement** (countering sycophancy through epistemic tension - AI deliberately challenges user views)
4. Agentic Transparency and Control
5. Expertise Democratization
6. Social-Emotional Augmentation
7. Duration-Optimized Integration (time-limited AI use to prevent full offloading)
