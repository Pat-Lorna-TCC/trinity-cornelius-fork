## Core Framework

Revolutionary framework treating different scales of a system as slices of a higher-dimensional object. Provides quantitative methods to calculate the causal contribution of each scale, showing what each uniquely adds beyond compression.

---

## Context & Acceptance

**Field Acceptance**: Emerging framework (2025), building on earlier causal emergence work

**Key Innovations**:
1. **Causal Apportioning Schema**: Calculates causal contribution of each scale
2. **Emergent Complexity Measure**: Quantifies how widely distributed causal workings are across hierarchy of scales
3. **Macroscale Causation**: Identifies cases where reduction to microscale is possible yet lossy about causation

---

## Cross-Level Causation Taxonomy

**Three Types of Causation**:

1. **Upward Causation**: Supervenience relation (macro effects from micro variables)
   - Example: Neural firing patterns → conscious experience

2. **Intra-Level Causation**: Causal effects within same scale
   - Example: Neuron-to-neuron signaling at micro level

3. **Downward Causation**: Macro-level properties influence micro-level properties
   - Example: Attention (macro) modulating neural processing (micro)
   - Example: Cultural norms (macro) shaping individual behavior (micro)

## Why This Matters

**Resolves Reductionism Debate**: Macro-level descriptions are not just convenient compression - they contain unique causal information

**Measurement Framework**: Provides quantitative tools to test emergence claims empirically

**Cross-Domain Application**: Applicable to consciousness, AI systems, organizations, ecosystems
