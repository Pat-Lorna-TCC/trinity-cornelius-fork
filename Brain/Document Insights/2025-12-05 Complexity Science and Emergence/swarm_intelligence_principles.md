## Core Framework

Swarm intelligence replaces a single complex controller with many simple agents. Each agent runs its own sense-decide-act loop. Communication is **local** (direct messages or shared signals), and **global behavior emerges** from repeated local updates across the swarm.

## Key Principles

**1. Local Communication Only**:
- Direct messages between nearby agents
- Shared environmental signals (pheromone maps, fields)
- No global coordination or broadcast

**2. Simple Individual Rules**:
- Each agent follows basic behavioral protocols
- No agent has full system knowledge
- Individual simplicity → collective complexity

**3. Emergent Global Behavior**:
- System-level patterns arise without central planning
- Coordination emerges from interaction dynamics
- Novel capabilities not present in individuals

**4. Scalability**:
- Add more agents without redesigning system
- Decentralized control enables massive scale
- Performance typically improves with size

**5. Robustness**:
- Failure of some agents degrades performance gradually
- No single point of failure
- Self-repair through redundancy

**Innovation**: No explicit communication - coordination through physics alone

**Mechanism**: Robots physically interact with load → mechanical coupling → emergent coordination
