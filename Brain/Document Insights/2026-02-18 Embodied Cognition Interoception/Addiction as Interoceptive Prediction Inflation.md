# Addiction as Interoceptive Prediction Inflation

## Core Insight

Addiction is a disorder of **interoceptive prediction calibration**. Drug cues trigger inflated body-level predictions of incoming reward - the interoceptive "hunger signal" for the drug is a genuine physiological experience, but it is systematically decoupled from the actual reward the drug delivers. Actual consumption produces an **attenuated dopamine increase** that cannot satisfy the inflated interoceptive expectation. This gap between bodily prediction and actual reward drives compulsive repetition. This is the Somatic Marker gone catastrophically wrong.

---

## The Mechanism

**Normal Reward Cycle**:
1. Cue → moderate interoceptive prediction of reward
2. Consumption → dopamine release matching prediction
3. Prediction error ≈ 0 → satisfaction
4. Behavior: normal motivation, normal cessation

**Addiction Cycle**:
1. Drug cue → **inflated** interoceptive prediction (body anticipates massive reward)
2. Drug consumption → **attenuated** dopamine increase (tolerance has reduced response)
3. **Gap** = interoceptive expectation >> actual reward
4. Large negative prediction error → compulsion to close the gap
5. Behavior: increased seeking, impaired satiation, reduced sensitivity to natural rewards

**The Somatic Marker Corruption**:
- The body's "hunger signal" for the drug is real and physiological - it is a genuine interoceptive experience
- But it is systematically decoupled from what the drug actually delivers
- The somatic marker now points toward a reward that the drug cannot actually provide
- This creates the phenomenology of craving: a bodily imperative pointing at an inadequate target

## Integration with Dopamine Framework

The existing Dopamine knowledge hub explains reward prediction error in the abstract. This framework provides the **interoceptive substrate** for why that prediction error is experienced as visceral craving:

**Dopamine account**: Prediction error drives behavior
**Interoceptive account**: The prediction error is felt in the body as an uncomfortable bodily state (craving, withdrawal) that demands action

Together: Addiction is not just a cognitive belief that drugs are rewarding. It is a bodily expectation system that generates real somatic discomfort (craving) that can only be temporarily resolved by consumption - and the resolution is inadequate, perpetuating the cycle.

**Reduced Sensitivity Finding**: People with addiction show "reduced sensitivity to changes in interoceptive signals" - the body has adapted to overshoot interoceptive expectations, so normal signals no longer register as meaningful. This explains anhedonia in addiction: natural rewards cannot generate sufficient interoceptive prediction to compete with drug cue-induced predictions.
