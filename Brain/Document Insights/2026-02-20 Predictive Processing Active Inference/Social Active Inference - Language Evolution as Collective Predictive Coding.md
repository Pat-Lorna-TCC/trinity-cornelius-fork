## Core Finding

**Language evolution as collective predictive coding**: Shared linguistic symbols emerge through decentralized, independent action decisions by individual agents each minimizing their own free energy. Language is not designed - it self-organizes from individual free energy minimization that includes modeling other agents.

**Mechanism:**
1. Individual agents minimize prediction error about their local environment, including other agents
2. Symbols that successfully reduce prediction errors (enable coordination) spread
3. Symbols that fail to reduce prediction errors (cause miscoordination) die out
4. Shared symbol systems = the precision-aligned generative models that emerge from this process

This is not a metaphor - it is a computational account of linguistic convention formation as free energy minimization at the collective level.

## Learning as Epistemic Foraging

The Di Paolo et al. (2024) paper reframes human learning as **epistemic foraging and prediction error minimization**:
- Humans don't learn by absorbing information - they learn by actively seeking prediction errors that update their generative models
- Passive information exposure is low-value (low prediction error = low learning signal)
- Active inquiry is high-value (seeking surprising information = high epistemic free energy = high learning signal)

**For AI integration in education**: Generative AI should NOT be a passive information dispenser (the current ChatGPT use case). It should be an **epistemic partner** - generating precision-enabling environments that stimulate productive prediction errors.

## Implications for AI Content and Communication

If communication = precision alignment between generative models, then effective communication is NOT about transmitting information. It is about:
- Generating prediction errors in the listener that update their model toward the communicator's model
- Using surprise strategically (novel framings, counter-intuitive claims) to create high-precision updating moments
- Building shared symbol systems (terminology, frameworks) that create precision alignment
