## Hypothesis

Flow states represent **precision redistribution** from self-referential priors (DMN-mediated) to action-relevant predictive loops. This is Bayes-optimal performance: all computational resources (precision) are allocated to the prediction-action cycle most relevant to the current task, with self-referential processing overhead minimized.

Flow is not merely "being in the zone" - it is the phenomenological experience of **Bayes-optimal cognition**: perfect calibration of confidence weights to task demands, with zero wasted precision on self-model maintenance.

---

## The FEP Account of Flow

**Normal consciousness:**
- High precision on self-model (DMN active = self-referential prediction running constantly)
- Cognitive resources split between task and self-narrative

**Flow state:**
- Precision shifts entirely to task-relevant action-perception loops
- Self-model precision collapses (experienced as ego dissolution or "being in the zone")
- Action-perception precision maximizes (experienced as effortless, fluent performance)
- Expert generative model accurately predicts task demands → surprise (prediction error) approaches zero

**Why flow feels effortless:** The generative model is so accurately tuned to task demands that minimal updating is required. Near-zero prediction error = near-zero effort. The brain is running perfectly calibrated predictions and barely needs to correct.
