# PFC Is a Priority Controller for Working Memory Not a Storage System

**Source**: "Prioritizing Working Memory Resources Depends on the Prefrontal Cortex", Hallenbeck, Tardiff, Sprague & Curtis, March 12, 2025, Journal of Neuroscience, 45(11), e1552242025. DOI: 10.1523/JNEUROSCI.1552-24.2025

## Core Insight

The prefrontal cortex (specifically the superior precentral sulcus, sPCS) does not store working memory representations - it controls the **differential allocation of mnemonic resources** across items held in mind. Visual cortex stores actual working memory content. The causal proof: disrupting sPCS with TMS paradoxically *improved* memory for low-priority items while degrading high-priority memory - exactly what happens when a priority controller is disabled (resources equalize across items).

---

## Evidence

**Study Design**: Causal TMS disruption study with concurrent fMRI
**Brain Regions Targeted**: Superior precentral sulcus (sPCS) vs. intraparietal sulcus (IPS2) as active control
**Paradigm**: Participants remembered two items (high- and low-priority, cued by probe frequency) during TMS

**Key Results**:
- TMS to sPCS disrupted normal prioritization of working memory resources
- **Counter-intuitive finding**: Disrupting sPCS *improved* memory for low-priority items (while degrading high-priority items)
- This is precisely what a resource-allocation control model predicts: disabling the controller equalizes resource distribution across items
- TMS to IPS2 (active control) showed no such effect
- Effect was hemifield-specific, confirming spatially precise, retinotopic control mechanism

**Interpretation**:
- sPCS = top-down allocation controller (not storage)
- Visual cortex = actual storage of working memory representations
- This resolves a decades-long debate between "PFC storage" and "PFC control" models

## Why This Matters

### Conceptual Reframe: PFC as Arbiter, Not Archive

The PFC does not "hold" information - it determines what gets preferential attention within working memory. This is a controller-vs-storage distinction with important implications:

- **Information loss under PFC disruption is selective, not general** - the PFC suppresses low-priority items while amplifying high-priority ones
- **PFC "damage" can paradoxically improve some cognition** - by releasing suppressed items from inhibition
- **Priority-setting is the key executive function** - not raw memory capacity
