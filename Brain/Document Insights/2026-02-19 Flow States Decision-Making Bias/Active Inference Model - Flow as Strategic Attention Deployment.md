## Core Framework

**Flow emerges from computational optimization of attention allocation.** The active inference framework (Friston et al., 2024) explains flow phenomenology as strategic deployment of precision weighting:

1. **High precision on immediate sensorimotor predictions**: Attention wholly focused on rapid task-specific demands
2. **Suppressed counterfactual planning**: No "stepping back" to consider alternatives
3. **Inhibited deep deliberation**: System 2 deliberation offline, only habitual System 1 operates

**Key Mechanism**: Flow achieves "the elimination of reflective self-awareness" while maintaining skilled agency by deploying attention exclusively to rapid task-specific demands that **only expertise can track**.

## Why This Framework Matters

**Explains Flow's Paradoxes**:

1. **"Effortless Effort"**: Arises because System 2 deliberation is offline - only habitual System 1 procedural knowledge operates. No deliberative processes competing for attention.

2. **Ego Dissolution + Agency**: You lose sense of self (no self-as-object modeling) but maintain skilled action (sensorimotor predictions still operating).

3. **Expertise Prerequisite**: Only experts can track rapid task fluctuations without deliberation. Novices require System 2 meta-cognitive monitoring, which prevents flow.

4. **Time Distortion**: No mental resources allocated to temporal monitoring (counterfactual planning suppressed).

5. **Intrinsic Motivation**: Predictive success itself is rewarding (minimizing prediction error is the brain's core drive).
