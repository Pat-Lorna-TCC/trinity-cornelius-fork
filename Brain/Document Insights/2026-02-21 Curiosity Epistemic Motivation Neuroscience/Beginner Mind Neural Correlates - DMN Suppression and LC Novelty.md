Buddhist "beginner's mind" (shoshin) - Shunryu Suzuki: "In the beginner's mind there are many possibilities, but in the expert's mind there are few" - has clear neural correlates emerging from 2024-2025 curiosity research. ## The Neural Mechanism of Beginner's Mind

**Step 1 - Expert's Mind = High-Precision Priors (Anti-Curiosity)**:
- The Default Mode Network (DMN) generates self-referential predictions, including expert-level prior beliefs about how things should be
- High DMN activity = high-precision priors = top-down dominance = low prediction error = low curiosity
- Expert's mind: "I already know how this works" → minimal prediction error → minimal dopamine → minimal curiosity

**Step 2 - Beginner's Mind = DMN Suppression (Pro-Curiosity)**:
- Awe and wonder states DEACTIVATE the DMN (established research; see vault notes on flow)
- DMN deactivation = reduced self-referential overlay = reduced top-down prediction dominance
- Reduced prior precision = more weight on incoming sensory data = more prediction error = more curiosity
- This is the neural mechanism by which beginner's mind generates genuine openness

**Step 3 - Curiosity Expression**:
- Reduced DMN → increased LC-NE activation (System 2 curiosity: perceptual/novelty)
- vmPFC encodes sensory uncertainty without top-down suppression → more curiosity signal
- Information gaps emerge naturally (priors not suppressing them) → System 1 curiosity fires
