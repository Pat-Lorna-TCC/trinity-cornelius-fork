## Core Insight

Curiosity doesn't just motivate information seeking - it literally shapes HOW the brain builds its spatial representations of the world. Curious individuals form richer, more interconnected hippocampal cognitive maps with better-linked landmarks. The same mechanism that drives knowledge curiosity also drives physical exploration and produces superior spatial memory. This connects curiosity directly to the Extended Mind framework: curiosity determines the richness of the external cognitive scaffolding the brain builds.

---

## Evidence

**Communications Psychology Study (2024)**:
- Curious individuals form richer cognitive maps with more interconnected landmarks
- Curiosity-driven exploration produces better-organized hippocampal spatial representations
- The mechanism: curiosity produces MORE exploration (not just faster movement), which creates more overlapping spatial memories and richer map integration

**Frontiers in Cognition (Children, 2024)**:
- Curiosity improves route memory in children specifically
- The curiosity effect on spatial memory occurs because curious children actively test spatial predictions ("let me see what's around this corner")
- This is System 1 curiosity (information gap: "I predict X is there; let me test") operating in the spatial domain

**Shared Hippocampal System**:
- The hippocampus supports BOTH episodic memory AND spatial navigation
- Curiosity-driven exploration and curiosity-driven memory encoding use the same hippocampal circuit
- This explains why curiosity literally "builds more interconnected memory structures" - it's the same system operating on both external space and internal knowledge space
