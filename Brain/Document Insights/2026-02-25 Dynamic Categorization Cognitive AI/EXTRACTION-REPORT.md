## Unique Insights Created

### 1. Dual-Prototype-Exemplar-Representations-Parallel-Neural-Substrates
**Core Insight**: Brain maintains both prototype (vmPFC/anterior hippocampus) and exemplar (IFG/lateral parietal) systems simultaneously for maximum flexibility

**Why Novel**: Resolves decades-long debate - answer is "both, in parallel" not "either/or"

**Connections**:
- Mental Models Taxonomy - Dual-system thinking validated
- Four-Type Memory Architecture - Semantic (prototypes) vs episodic (exemplars)
- AI hybrid approaches - Biological validation

**Source**: Davis et al., *eLife* (2020, highly cited 2024-2025)

---

### 2. Progressive-Abstraction-Exemplar-to-Prototype-Shift-During-Learning
**Core Insight**: Learning trajectory moves from exemplar (specific instances) → prototype (generalized) representations as expertise develops

**Why Novel**: Dynamic process not static choice; mirrors memory consolidation

**Connections**:
- Memory Consolidation - Episodic to Semantic Pathways - Same process
- Zettelkasten workflow - Fleeting → Permanent notes
- Few-shot → Zero-shot learning

**Source**: *Journal of Intelligence* 14(6):470 (June 2024)

---

### 3. Gated-Attention-Improves-Categorization-Through-Selective-Feature-Filtering
**Core Insight**: Sigmoid gating after attention improves categorization by suppressing irrelevant features, amplifying diagnostic ones

**Why Novel**: NeurIPS 2025 Best Paper; production-deployed (Qwen3-Next); simple modification, major impact

**Connections**:
- Attention as Universal Selection Pressure - Mechanism at feature level
- Mental Models Taxonomy - Lenses as selective filters
- Agent tool selection - Feature-based routing

**Source**: Alibaba Qwen team, NeurIPS 2025 Best Paper, arXiv:2505.06708
