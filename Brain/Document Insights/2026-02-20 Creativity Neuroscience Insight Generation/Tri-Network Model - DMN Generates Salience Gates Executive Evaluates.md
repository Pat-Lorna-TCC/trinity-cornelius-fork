# Tri-Network Model - DMN Generates, Salience Gates, Executive Evaluates

**Source**: "Neural dynamics during the generation and evaluation of creative and non-creative ideas" - Communications Biology - 2025 - https://www.nature.com/articles/s42003-025-09018-3

## Core Framework

The creative process follows a **temporal three-stage sequence** with distinct neural dynamics at each phase:

1. **DMN (Default Mode Network)** - Generates ideas spontaneously through network reconfiguration
2. **Salience Network (SN)** - Acts as gatekeeper/router, identifying promising candidates
3. **Executive Control Network (ECN)** - Evaluates, refines, and selects ideas through network integration

**Critical distinction**: Generation uses **reconfiguration**; evaluation uses **integration**.

---

## Evidence

**Study Design**: Within-subject fMRI with repeated measures, two sessions one week apart
**Tasks**: Creative vs. non-creative idea generation and evaluation (within-subject design provides clean comparison)

**Key Results**:
- Generating creative ideas produces significantly higher **NETWORK RECONFIGURATION** than non-creative generation
- Evaluating creative vs. non-creative ideas produces similar **NETWORK INTEGRATION** (not reconfiguration)
- The Salience Network (specifically right anterior insula) acts as filter/router between DMN and ECN
- Network switching characterizes GENERATION; network integration characterizes EVALUATION
- First within-subject demonstration of different neural dynamics for the two phases

---

## The Salience Network as Creative Router

**Previously underemphasized**: The Salience Network's role in creativity was less studied than DMN and ECN.

**New understanding**: The Salience Network (right anterior insula + anterior cingulate cortex) acts as:
- A **relevance filter** - identifies which spontaneous DMN-generated ideas have enough promise to forward to ECN
- A **routing mechanism** - determines what gets evaluated and what gets discarded
- A **gating system** - without this gate, all ideas would either stay as unconstrained daydreams OR flood ECN overwhelmingly

**Why this matters for creativity**:
- Without Salience Network gating, ideas never leave the DMN's spontaneous generation
- With too tight Salience Network filtering, only conventional ideas pass to ECN
- Optimal creativity requires **calibrated** Salience Network sensitivity - loose enough to let unusual ideas through, tight enough to filter noise

## Temporal Sequence Model

```
PHASE 1: GENERATION (DMN dominant)
- Network reconfiguration (high flexibility)
- Spontaneous associations arise
- Multiple competing idea fragments form
- Characterized by high entropy/variability

PHASE 2: SALIENCE DETECTION (Salience Network active)
- Right anterior insula identifies "interesting" candidates
- Acts as first-pass filter on generative output
- Routes promising fragments toward ECN
- Discards noise and redundant associations

PHASE 3: EVALUATION (ECN dominant)
- Network integration (high coherence)
- Ideas assessed for novelty AND appropriateness
- Selection and refinement occurs
- Characterized by low entropy/high coherence
```

## Implications for Creative Practice

**Understanding creative blocks**:
- **Block at Phase 1** (generation): DMN underactive → incubation, mind-wandering needed
- **Block at Phase 2** (routing): Salience Network too tight → ideas generated but not forwarded → feel "blank" despite mind activity
- **Block at Phase 3** (evaluation): ECN overcritical → ideas arrive but get rejected too quickly → perfectionism block

**Practical protocols based on this model**:
- **Brainstorming**: Maximize Phase 1 (DMN), defer Phase 3 (ECN) - the "yes-and" rule
- **Incubation**: Let Phase 1 run unconsciously, Phase 2 curates before conscious awareness
- **Editing**: Engage Phase 3 fully AFTER Phase 1 is complete - don't mix
- **Flow state**: Optimal when all three phases are synchronized in real-time

## AI Architecture Parallel

This tri-network model maps directly onto AI agent design principles:

- **DMN → Generative/exploration module**: Produces candidate ideas/actions without evaluation
- **Salience Network → Relevance/prioritization layer**: Filters candidates by estimated value
- **ECN → Evaluation/selection module**: Applies criteria to select and refine
