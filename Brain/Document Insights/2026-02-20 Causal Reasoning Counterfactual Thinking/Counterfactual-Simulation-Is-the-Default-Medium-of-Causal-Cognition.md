# Counterfactual Simulation Is the Default Medium of Human Causal Cognition

## Core Insight

Human causal reasoning does not primarily use logical rule-based inference. Instead, it uses **mental simulation** - the hippocampus provides episodic "building blocks" that are recombined into novel counterfactual scenarios. This constructive simulation is the brain's default mode of causal cognition. TMS disruption of hippocampal activity reduces both future event simulation AND creative thinking, confirming they share the same constructive mechanism.

This reveals why humans can reason causally about entirely novel scenarios - they simulate rather than calculate.

---

## The Hippocampal Recombination Mechanism

**Key mechanism**: Episodic memories are stored as separable elements in the hippocampus. When constructing a counterfactual ("What if X had not happened?"), the brain:
1. Retrieves relevant episodic building blocks from memory
2. Recombines them in a novel configuration that removes or modifies the target event
3. Runs a forward simulation through the recombined scenario
4. Generates an outcome prediction for the counterfactual world

This is why **hippocampal damage** impairs not only past memory but also:
- Future planning (episodic future thought)
- Counterfactual reasoning ("what if?")
- Creative imagination
- Spatial navigation in novel environments

All four are expressions of the same hippocampal recombination engine.

## Evidence

**TMS Study (PNAS)**: Direct disruption of hippocampal activity using transcranial magnetic stimulation reduces:
- Episodic future simulation (imagining future events)
- Divergent creative thinking
Confirming a shared constructive mechanism - not separate capacities.

**Narrative Comprehension (Trends in Cognitive Sciences, 2024)**:
- Default Mode Network regions (MTL, hippocampus) show causal structure-dependent activity during narrative comprehension
- Brain states during causal integration moments are **synchronized across individuals** - suggesting a shared neural algorithm for cause-effect parsing
- Causally connected events are remembered better than temporally connected events without causal links

## The Distinction Between Causal and Counterfactual Explanations

A 2025 *Thinking and Reasoning* paper by Byrne (Trinity College Dublin) revealed that causal and counterfactual explanations are **computationally distinct** - they focus on different aspects of the same event:

- **Causal explanation** ("Why did X happen?"): Focuses on actual antecedents that brought about the event
- **Counterfactual explanation** ("What if X hadn't happened?"): Focuses on mutable features that could have prevented the event

For exceptional events, people focus on the exceptional element when constructing counterfactuals, but shift to *controllable choices* when constructing causal explanations. This dissociation reveals two distinct cognitive processes running on the same hippocampal simulation engine.

## The Recency Effect in Causal Attribution - A Simulation Mechanism

A 2025 *Cognition* computational model showed that the recency effect in causal attribution (recent causes feel more causal) is mediated through differential willingness to simulate counterfactuals for recent events - NOT through direct temporal sensitivity.

**Insight**: People are more willing to imaginatively "undo" recent events than distant ones, so recent events generate stronger counterfactuals, which in turn makes them feel more causally potent. Temporal order affects causality through the simulation pathway.

This explains Kahneman's "norm theory" (unusual events feel more causal) as a simulation accessibility effect - it's easier to simulate a world where the unusual event didn't happen.

## Why LLMs Lack This Capacity

LLMs lack:
1. An episodic memory store with recombineable elements
2. A forward simulation engine for running counterfactual scenarios
3. The hippocampal-DMN integration that creates coherent possible-world representations

They can retrieve text describing counterfactuals from training data but cannot construct genuinely novel counterfactual scenarios through simulation. This is the computational gap that explains the benchmark failures in LLMs Are Causal Parrots - Stuck at Rung-1.
