## Synthesis

The 2025 research findings enable a more complete model of how attention, dopamine, and flow interact:

**The Motivational Attention Architecture**:

```
Dopamine system → Configures motivational state → Determines attention quality
       ↓
High motivation → Facilitation loop dominant → Sustained effortful attention
Low motivation → Inhibition loop dominant → Directed attention fatigue
       ↓
Task automaticity + Intrinsic reward → Flow state
       ↓
Flow = Effortless sustained attention (facilitation loop at peak, no depletion)
```

This model integrates:
1. Dopamine as motivational configurator (Cell 2025)
2. Facilitation vs. inhibition loop dynamics (Brain Communications 2025)
3. Flow as the effortless endpoint (Scientific Reports 2025)

---

## Rationale

**Three-Stage Progression**:

**Stage 1: Motivated Engagement** (effortful, sustainable for 90-120 min BRAC cycles)
- Dopaminergic arousal-reward network active
- Facilitation loop (thalamic-striatal-frontal) dominant
- Directed attention maintained through inhibitory effort
- Depletes over time → fatigue = inhibition loop takes over

**Stage 2: Directed Attention Fatigue** (the wall)
- Inhibition-limbic system overwhelms facilitation loop
- Dopamine, serotonin, acetylcholine depleted
- Restoration required (nature, movement, sleep)
- Willpower cannot override the network state shift

**Stage 3: Flow** (effortless, potentially indefinite duration)
- Automaticity replaces inhibitory effort
- Intrinsic reward loop (dopaminergic striatal) sustains attention
- mPFC suppressed (no self-referential processing = no mind-wandering)
- Facilitation loop maintained without depletion because no inhibitory work required

**Confidence Level**: Medium - individual components are empirically supported; the integrative model is a synthesis

## Testable Predictions

1. Individuals with higher baseline dopamine function should show longer directed attention periods before fatigue
2. Tasks that activate intrinsic reward loops should show longer sustained attention than extrinsically motivated tasks at equivalent difficulty
3. Flow states should show neurological markers that distinguish the "no-depletion" mechanism from merely-high effort states
4. Restoration interventions that specifically reset the dopamine-motivation system (not just reduce arousal) should be more effective than passive rest
