## Core Insight

A 2024 CLT update makes a critical distinction that resolves apparent contradictions in multimedia learning research: **modal redundancy** (images + text in the same modality) actually FACILITATES learning and lowers cognitive load, while **codal redundancy** (spoken + written text = same linguistic code via two channels) INCREASES load through the classic "redundancy effect." The two types of redundancy have opposite effects.

---

## Evidence

**Trypke et al. (2024)**:
- **Modal redundancy**: Images + text = two representations in the SAME modality (e.g., visual)
  - Effect: FACILITATES learning, lowers cognitive load
  - Mechanism: Multiple representations of the same material strengthen schema formation
- **Codal redundancy**: Spoken text + written text = same LINGUISTIC CODE in two modalities
  - Effect: Creates classic redundancy effect, INCREASES load
  - Mechanism: Two identical linguistic representations compete for the same processing channel

**CLAM Framework (MDPI, 2025)** - Extended CLT now incorporates:
- Biometric monitoring (EEG, eye tracking, pupillometry) for real-time cognitive load measurement
- Adaptive AI systems that modify content based on detected load
- Affective variables (anxiety, interest) as moderators
- Neurodiverse populations requiring differentiated approaches

**CLT Critical Perspective (Educational Philosophy and Theory, 2024)**:
- CLT's insistence on direct instruction opposes inquiry-based learning
- Productive struggle (germane load) may be eliminated by over-optimized instruction

---

## Practical Design Implications

This distinction resolves a common confusion in content creation:
- **Captions + video visuals**: Modal redundancy → Helpful (same modality, different representation)
- **Read-along captions on narrated video** (identical text + speech): Codal redundancy → Harmful (same linguistic code, two channels)
- **Diagram + explanatory text**: Modal redundancy → Helpful
- **Bullet points + narrator reading bullets verbatim**: Codal redundancy → Harmful

**For AI-generated educational content**: The distinction means AI should not simultaneously narrate and display identical text. Complementary representations (visual + explanatory text) enhance; identical representations (spoken = written) impede.
