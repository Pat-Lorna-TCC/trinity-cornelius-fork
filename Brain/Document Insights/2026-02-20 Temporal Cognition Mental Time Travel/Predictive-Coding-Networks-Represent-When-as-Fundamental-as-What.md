# Predictive Coding Networks Represent When as Fundamental as What

## Core Insight

Temporal predictive coding networks can represent WHEN predictions jointly with WHAT predictions, with temporal prediction being just as fundamental as content prediction for neural processing. When trained on natural dynamic inputs, these networks produce motion-sensitive receptive fields resembling V1 neurons - suggesting the visual cortex itself is fundamentally a temporal prediction system, not a static feature detector.

The key finding: the brain does not first detect "what" and then separately process "when" - temporal structure is processed in parallel with content from the earliest processing stages.

---

## Evidence

**PLOS Computational Biology (Millidge et al., Oxford, 2024)**:
- Temporal predictive coding networks can approximate Kalman filter performance for linear systems
- Achieved with biologically plausible local computations only - no complex mathematical operations required
- When trained on natural dynamic inputs: generates Gabor-like, motion-sensitive receptive fields resembling V1 neurons
- "When" predictions are as fundamental as "what" predictions

**Predictive Coding Light (PCL, Nature Communications 2025)**:
- Recurrent hierarchical spiking neural network
- Suppresses the most predictable spikes, transmits compressed representations
- Uses biologically plausible spike-timing learning rules
- Reproduces wealth of visual cortex findings

**Dynamic Predictive Coding (PLOS Computational Biology, Feb 2024)**:
- When coupled with associative memory emulating hippocampal function, allows episodic memories to be stored and retrieved via cue-triggered recall
- Directly connects predictive coding framework to episodic future thinking

## Theoretical Significance

This finding extends the predictive processing framework (which the vault references in terms of belief updating and prediction error) to temporal cognition specifically:
- The brain doesn't just predict WHAT will happen - it predicts WHEN each part of the sequence will occur
- Prediction errors have both content and temporal components
- The integration of hippocampal episodic memory with predictive coding creates a system capable of cue-triggered recall of temporal sequences - the computational substrate for mental time travel
